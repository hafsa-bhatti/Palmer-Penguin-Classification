# Palmer-Penguin-Classification
Python scripts to clean, explore, and classify Palmer Penguins ecological dataset using Pandas, Seaborn, Matplotlib and Scikit-learn.
# Purpose

Demonstrate a real-world data science workflow, cleaning, EDA and machine learning, applied to the Palmer Penguins dataset to predict species from physical measurements.

# Tech Used

Python
Pandas
NumPy
Matplotlib 
Seaborn
Scikit-learn
